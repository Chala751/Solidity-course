# Solidity Course Repository

This repository contains the group work for the Alchemy University Learn Solidity course.

## Structure

- materials/ - course handouts, slides, and other provided resources
- code/ - exercises, projects, and code written during the course
- screenshoots/ - screenshots proving course progress and completion

## Contribution Rules

- Everyone must make at least one public commit.
- Keep files organized in the correct folder.
- Use clear commit messages (e.g., "Add lesson 2 contract", "Add progress screenshot").
