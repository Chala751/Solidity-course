# Solidity Functions Practice

## Lessons

- Arguments (Arguments.sol)
- Increment (Increment.sol)
- View Addition (ViewAddition.sol)
- Console Log (ConsoleLog.sol)
- Pure Double (PureDouble.sol)
- Double Overload (DoubleOverload.sol)

## Required Installation

Option 1: Use Remix (no local install)
- Open https://remix.ethereum.org/
- Create the .sol file and paste the code
- Compile with the Solidity compiler version 0.8.20

Option 2: Compile locally with solc
- Install Node.js (LTS) from https://nodejs.org/
- From this folder, run:
  - npm init -y
  - npm install --save-dev solc
- Compile:
  - npx solc --version
  - npx solc Arguments.sol
